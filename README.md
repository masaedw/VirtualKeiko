# virtualkeiko

A virtual keiko server

http://virtualkeiko.herokuapp.com/

## Prerequisites

[Leiningen][1] 2.0.0 or above.

[1]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

    lein ring server
